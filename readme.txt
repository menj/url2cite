== Description ==
Professional citation generator with:
- 10+ citation styles (APA, MLA, Chicago, etc.)
- Auto-detection of URLs in content
- Schema.org microdata support
- REST API integration
- Caching system for performance

== Installation ==
1. Upload the plugin via WordPress admin
2. Activate and configure in Settings → URL2Cite
3. Use shortcodes/blocks to insert citations

== Frequently Asked Questions ==
= How to clear cached data? =
Go to Settings → URL2Cite → Click "Clear Cache".

= Supported citation styles? =
APA, MLA, Chicago, Harvard, IEEE, Vancouver, Turabian, Bluebook, CSE, Wikipedia.
